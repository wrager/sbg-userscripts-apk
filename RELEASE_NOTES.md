# v0.9.0

## Автопровизия скриптов

- Скрипты из пресетов автоматически загружаются и включаются при первом запуске
- Анимированный индикатор загрузки во время автопровизии

## Стартовый экран

- Удалён стартовый экран — приложение сразу открывает игру

## Менеджер скриптов

- Переработаны карточки: компактный вид, двухстрочная секция деталей, конфликты и статус актуальности в строке деталей
- Настройки перенесены в выдвижную панель с pull-tab
- Кнопка-меню карточки перемещена в строку названия
- Удалены: кнопка «Update all», автопроверка при открытии, Toast проверки обновлений
- Кнопка «Добавить скрипт» временно отключена

## WebView

- Исправлена потеря window event listeners при document.write()
- Исправлено поведение экранной клавиатуры
- Кнопка «Перезагрузить игру» теперь возвращает на начальный URL

## Прочее

- Удалён скрипт SBG plus (Anmiles) из пресетов
- Anti-cache заголовки при загрузке скриптов
- Новая схема версионирования (versionCode вычисляется из компонентов версии)
