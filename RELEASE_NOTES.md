# v0.11.0

## Автообновления

- Приложение проверяет обновления при запуске — диалог с release notes и возможностью скачать и установить APK
- Скрипты проверяются при запуске — диалог со списком доступных обновлений и кнопкой «Обновить все»

## Синхронизация с настройками игры

- Тема и язык приложения автоматически подстраиваются под настройки игры

## Прочее

- Настройка «Не гасить экран»
- Кнопка «Назад» закрывает drawer и менеджер скриптов
