# v0.4

## Менеджер скриптов

- Переработан UI: единый экран загрузки и включения скриптов
- Загрузка скриптов по требованию вместо автозагрузки
- Прогресс загрузки скриптов с анимацией ProgressBar
- Проверка обновлений скриптов с отображением статуса
- Кнопка «Обновить» вместо текста «Доступно обновление»
- Разрешено удаление пресетных скриптов
- Кнопка «Назад» в менеджере скриптов
- Экран скриптов показывается только при первом запуске

## Настройки

- Добавлена функция перезагрузки игры при изменении скриптов
- Опция проверки обновлений при открытии менеджера

## Исправления

- Fullscreen режим: корректная работа через onWindowFocusChanged
- Перекрытие WebView системными барами в неполноэкранном режиме
- Сохранение cookies при сворачивании игры
- Определение конфликтов скриптов
- Краш MaterialSwitch при измерении layout
