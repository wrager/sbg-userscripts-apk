# v0.12.0

- **Release notes скриптов.** При обновлении скриптов показываются release notes с GitHub — в диалоге обновлений и в менеджере скриптов.
- **Диагностика баг-репортов.** Кнопка «Сообщить об ошибке» собирает информацию об устройстве, версии приложения и WebView, списке скриптов и JS-ошибках, копирует в буфер обмена и открывает GitHub Issues с предзаполненными данными.
- **Обновлённая иконка** — добавлен текст «SBG» над «SCOUT».
